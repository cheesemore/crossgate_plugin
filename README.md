# crossgate_plugin

《魔力宝贝：序章》本地工具链：热补丁 GUI、多开器、助手（及共享库）。

> **不含游戏客户端**（`cg37.exe` / `cg37_Data` 等）。请在本机已安装的游戏目录旁使用，或在补丁 GUI 中选择游戏根目录。

## 目录

| 路径 | 说明 |
|------|------|
| `魔力宝贝序章补丁/` | 热补丁 GUI / 组合打补丁 / 桥接注入 |
| `序章多开器/` | 多账号启动与绑定 |
| `魔力宝贝序章助手/` | 单实例助手 GUI |
| `序章多开助手/` | 多开相关助手脚本 |
| `序章助手共享/` | 助手公共逻辑 |
| `tools/hotfix_patcher/` | C# 补丁引擎源码 |
| `tools/seqchapter_nine_action/` | 神奇九动 DLL 源码 |
| `tools/seqchapter_helper_bridge/` | 助手桥接 DLL 源码 |

## 快速开始

1. 关闭游戏 `cg37.exe`
2. 运行 `魔力宝贝序章补丁/启动补丁GUI.bat` → **初始化** → 勾选补丁 → **应用补丁**
3. 多开：`序章多开器/启动多开器.bat`
4. 助手：`魔力宝贝序章助手/启动序章助手.bat`

首次使用或改引擎后，在补丁目录运行 `重建补丁引擎.bat`（需 .NET SDK）。

维护说明见 `魔力宝贝序章补丁/补丁维护.md`。
